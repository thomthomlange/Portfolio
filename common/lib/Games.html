 La sécurité des passagers utilisant un moyen de transport repose sur la fiabilité de l'assemblage du moyen de transport. Afin de réduire les risques d'erreurs humain concernant ces assemblages, le projet Desecora a été créé.
L'assemblage de pièces boulonnées fait partie du processus de fabrication de différents moyens de locomotion: Train, Avion, Voiture, ... L'utilisation de ces moyens de transport implique la présence de vibrations qui vont desserrer et fragiliser l'assemblage. C'est pourquoi il existe des protocoles de sécurité de montage de pièces, impliquant une précision dans le torque appliqué, ainsi que dans l'ordre des boulons serrés.
Le logiciel va donc utiliser la réalité augmentée afin de vérifier et valider le serrage des pièces, et le respect des protocoles de serrage à l'aide d'une communication entre le casque ainsi qu'un outil de serrage connecté.
Le logiciel a été développé pour être utilisé sur un casque de réalité augmentée "Magic Leap 2", dont le développement a été effectué sur Unity.

 Le joueur se retrouve coincé dans un donjon dont il doit s'échapper.Le jeu commence par une section "Tuto", où le joueur rencontrera un certain Monsieur Patrick, qui va guider le joueur vers la sortie du donjon en lui expliquant comment fonctionne le donjon. Le joueur apprendra comment bouger, utiliser les objets trouvés, et réussira peut-être à sortir du donjon ...

Le jeu se joue sur clavier :

  • ZQSD ou les flèches directionnelles pour bouger
  • E pour interagir avec les éléments du décor
  • A pour utiliser les objets ramassés (Pour voir les effets des objets, le joueur doit attendre quelque seconde devant l'objet voulu afin de pouvoir voir un descriptif).
  • TAB pour changer l'ordre d'utilisation des objets
  • F pour utiliser la lampe torche
  • ECHAP pour ouvrir le menu pause du jeu
Tous les modèles 3D et 2D (sauf modèles utilisés sur les personnages) ont été créé pour le jeu.

 Lamb Drifters est un jeu d'action dans lequel je m'occupe du développement entier du jeu.

2 modes de jeu sont fonctionnels, un premier jeu qui se base sur un modèle "Boss Rush" où le joueur doit battre chaque boss afin d'arriver à la fin du jeu. Un deuxième mode de jeu, où le joueur doit battre autant de vagues de monstres que possible.

Le jeu se joue sur clavier et souris:

  • ZQSD ou les flèches directionnelles pour bouger
  • Clique Gauche de la souris pour attaquer
  • Clique Droit de la souris pour contrer
  • SHIFT pour déclencher le drift
  • ESPACE pour faire un dash
  • ECHAP pour le menu pause

Le mode multijoueur est aussi disponible sur les dernières version du jeu

 Voici un jeu vidéo réalisé durant 48h, pendant la Global Game Jam de 2025 avec comme thème "Bubbles". Projet créé sur unity, par un groupe de 4 étudiant de la promo 7 du master AMINJ de l'INU Champollion.
Dans ce petit projet, je me suis occupé des collisions, ainsi que les dégats de collisions reçu par le véhicule. j'ai notamment participé au level design, ainsi que le peaufinage du jeu (particules, sfx).
Le but du jeu est de dévaler la pente, dans l'idée de reproduire le style caisse à savon, avec un petit twist : La route est faite de savon !
Adieu la maniabilité, et bonjour le skill. Chaque contact à haute vitesse contre un obstacle détruira le véhicule petit à petit (sauf si une des pièces principales du véhicule est détruit, dans ce cas, le joueur aura automatiquement perdu).
Le jeu se joue sur clavier :

  • W et C pour diriger le véhicule à gauche ou à droite
  • Les flèches directionnelles pour pencher le personnage : la flèche du haut pour gagner en vitesse, et la flèche du bas pour en perdre.
  • R pour Reset

ou sur Manette :
  • Le joystick gauche pour diriger le véhicule à gauche ou à droite
  • Le joystick droit pour pencher le personnage : vers le haut pour gagner en vitesse, et le bas pour en perdre.

: 1 personne
 Jeu sérieux réalisé afin d'apprendre aux joueurs la position et les notes sur le manche d'une guitare.
Le joueur doit avancer dans un donjon qui se génère procéduralement, frapper les notes présentes dans la salle correspondant aux notes sur le manche de la guitare. Plus le joueur finit des niveaux, plus le nombre de salle augmente. A chaque niveau passé, le joueur regagne toute sa vie perdu.
Le joueur finit un niveau lorsqu'il trouve la guitare en couleur, et le jeu se termine au bout de 5 niveaux passé. A la fin du jeu, le joueur aura un retour sur le temps passé sur le jeu, le nombre d'erreur fait, et le nombre de Game Over du joueur pendant sa session.
Le jeu se joue sur clavier :

  • ZQSD ou les flèches directionnelles pour bouger
  • TAB pour agrandir l'image des notes à trouver
  • ECHAP pour ouvrir le menu pause
